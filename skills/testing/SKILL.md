# Testing

Use this skill when a role should prioritize behavioral verification, targeted regression coverage, and test-first implementation loops.

## Runtime Focus

- Reproduce the behavior with a failing focused test before changing production code.
- Verify both projection contracts and user-visible cues when runtime-facing data changes.
- Prefer focused package-level verification before broad repo-wide checks in a dirty workspace.
