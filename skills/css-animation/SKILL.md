# CSS Animation

Use this skill when a role should add meaningful UI motion or interaction polish without drifting into noisy animation.

## Runtime Focus

- Use motion to reinforce hierarchy, state change, and orientation rather than decoration.
- Keep transitions consistent with the existing dashboard shell and avoid heavy continuous animation.
